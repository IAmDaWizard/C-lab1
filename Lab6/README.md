## Постановка задачи

Необходимо написать bash-скрипт, который рекурсивно обходит указанный каталог и все его подкаталоги, находит текстовые файлы и кодирует их с помощью шифра Цезаря.

Для преобразования текста используется утилита `tr`.

## Описание лабораторной работы

В ходе выполнения лабораторной работы был реализован bash-скрипт `caesar.sh`.

Скрипт принимает два аргумента:

1. путь к каталогу;
2. величину сдвига для шифра Цезаря.

После запуска скрипт ищет все файлы с расширением `.txt` в указанной папке и во всех вложенных папках. Каждый найденный файл обрабатывается с помощью утилиты `tr`: буквы заменяются на другие буквы того же алфавита со сдвигом на заданное число позиций.

Например, при сдвиге `3`:

```text
a -> d
b -> e
c -> f
```

Для русского алфавита работает такая же логика:

```text
а -> г
б -> д
в -> е
```

Скрипт поддерживает английский и русский алфавиты, а также буквы верхнего и нижнего регистра.

## Код скрипта

```bash
#!/bin/bash

DIR="$1"
SHIFT="$2"

if [ -z "$DIR" ] || [ -z "$SHIFT" ]; then
    echo "Usage: $0 <directory> <shift>"
    exit 1
fi

SHIFT_EN=$((SHIFT % 26))
SHIFT_RU=$((SHIFT % 33))

LOWER_EN="abcdefghijklmnopqrstuvwxyz"
UPPER_EN="ABCDEFGHIJKLMNOPQRSTUVWXYZ"

LOWER_RU="абвгдеёжзийклмнопрстуфхцчшщъыьэюя"
UPPER_RU="АБВГДЕЁЖЗИЙКЛМНОПРСТУФХЦЧШЩЪЫЬЭЮЯ"

LOWER_EN_SHIFTED="${LOWER_EN:$SHIFT_EN}${LOWER_EN:0:$SHIFT_EN}"
UPPER_EN_SHIFTED="${UPPER_EN:$SHIFT_EN}${UPPER_EN:0:$SHIFT_EN}"

LOWER_RU_SHIFTED="${LOWER_RU:$SHIFT_RU}${LOWER_RU:0:$SHIFT_RU}"
UPPER_RU_SHIFTED="${UPPER_RU:$SHIFT_RU}${UPPER_RU:0:$SHIFT_RU}"

find "$DIR" -type f -name "*.txt" | while read file; do
    tr "$LOWER_EN$UPPER_EN$LOWER_RU$UPPER_RU" "$LOWER_EN_SHIFTED$UPPER_EN_SHIFTED$LOWER_RU_SHIFTED$UPPER_RU_SHIFTED" < "$file" > "$file.tmp"
    mv "$file.tmp" "$file"
done
```

## Как работает скрипт

Сначала скрипт получает два аргумента:

```bash
DIR="$1"
SHIFT="$2"
```

`DIR` — это папка, которую нужно обработать.  
`SHIFT` — это число, на которое нужно сдвигать буквы.

Далее выполняется проверка:

```bash
if [ -z "$DIR" ] || [ -z "$SHIFT" ]; then
    echo "Usage: $0 <directory> <shift>"
    exit 1
fi
```

Если пользователь не передал папку или сдвиг, скрипт выводит подсказку и завершает работу.

Затем сдвиг приводится к размеру алфавита:

```bash
SHIFT_EN=$((SHIFT % 26))
SHIFT_RU=$((SHIFT % 33))
```

Для английского алфавита используется 26 букв, для русского — 33 буквы. Это нужно, чтобы при большом сдвиге буквы корректно возвращались в начало алфавита.

Например, если сдвиг равен `44`, то для английского алфавита фактически используется:

```text
44 % 26 = 18
```

После этого задаются обычные алфавиты:

```bash
LOWER_EN="abcdefghijklmnopqrstuvwxyz"
UPPER_EN="ABCDEFGHIJKLMNOPQRSTUVWXYZ"

LOWER_RU="абвгдеёжзийклмнопрстуфхцчшщъыьэюя"
UPPER_RU="АБВГДЕЁЖЗИЙКЛМНОПРСТУФХЦЧШЩЪЫЬЭЮЯ"
```

Затем создаются сдвинутые алфавиты:

```bash
LOWER_EN_SHIFTED="${LOWER_EN:$SHIFT_EN}${LOWER_EN:0:$SHIFT_EN}"
UPPER_EN_SHIFTED="${UPPER_EN:$SHIFT_EN}${UPPER_EN:0:$SHIFT_EN}"

LOWER_RU_SHIFTED="${LOWER_RU:$SHIFT_RU}${LOWER_RU:0:$SHIFT_RU}"
UPPER_RU_SHIFTED="${UPPER_RU:$SHIFT_RU}${UPPER_RU:0:$SHIFT_RU}"
```

Если сдвиг равен `3`, английский алфавит:

```text
abcdefghijklmnopqrstuvwxyz
```

превращается в:

```text
defghijklmnopqrstuvwxyzabc
```

После этого команда `find` ищет все текстовые файлы:

```bash
find "$DIR" -type f -name "*.txt"
```

Для каждого найденного файла выполняется команда `tr`:

```bash
tr "$LOWER_EN$UPPER_EN$LOWER_RU$UPPER_RU" "$LOWER_EN_SHIFTED$UPPER_EN_SHIFTED$LOWER_RU_SHIFTED$UPPER_RU_SHIFTED"
```

Она заменяет каждый символ из обычного алфавита на соответствующий символ из сдвинутого алфавита.

Результат сначала записывается во временный файл:

```bash
> "$file.tmp"
```

После этого временный файл заменяет исходный:

```bash
mv "$file.tmp" "$file"
```

Так исходный текстовый файл становится зашифрованным.

## Запуск программы

Сначала нужно дать скрипту права на выполнение:

```bash
chmod +x caesar.sh
```

После этого скрипт можно запустить:

```bash
./caesar.sh test_folder 3
```

Здесь:

- `test_folder` — папка с текстовыми файлами;
- `3` — величина сдвига.

Можно использовать любой другой сдвиг:

```bash
./caesar.sh test_folder 5
```

или:

```bash
./caesar.sh test_folder 44
```

## Особенности реализации

Скрипт изменяет файлы на месте. Это значит, что исходный текст заменяется зашифрованным.

Чтобы не повредить файл во время обработки, результат сначала записывается во временный файл с расширением `.tmp`, а затем этот временный файл заменяет исходный файл.

Скрипт обрабатывает только файлы с расширением `.txt`. Остальные файлы не изменяются.
