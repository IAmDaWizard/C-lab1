# C-lab1

Laboratory works in C focused on algorithms, trees, sorting, graph algorithms and basic systems programming.

## About

This repository contains C programming labs completed as part of university practice. The main focus is on algorithms, data structures and manual memory management.

## Technologies

- C

- Shell scripts

## Topics covered

- Sorting algorithms
- Binary search
- Trees
- Graph algorithms
- Working with arrays and pointers
- Dynamic memory management
- Modular project structure in C

## Build and run

Clone the repository:

```bash
git clone https://github.com/IAmDaWizard/C-lab1.git
cd C-lab1
```

Build with CMake:

```bash
mkdir build
cd build
cmake ..
make
```

Run the needed executable from the build directory.
